    python不完整的基础

    与或非优先级

    列表

    
shopping_list = []
shopping_list.append("显示器")   #往列表添加值
print(shopping_list)

    列表可以放不同类型的数据（同许多语言不一样），自由灵活

    list1 =['Hello"]
list1.append(66.6)
list1.append(True)
list1.append(None)
print(list1)
#["Hello"，66.6，True, None]

    len()获取列表长度

    l利用索引来获得某个位置的元素

    lisit1[1]
    列表相关函数
    price =[799，1024，200，800]
max_price = max(price)   #获取列表最大值
min_price = min(price)
sorted_price = sorted(price)            #返回一个排序好的列表，不改变原来的列表
print(max_price)
print(min_price)
print(sorted_price)

    字典

    键值对

    contacts = {"小明":"13700000000" , "小花":"13700000001"}

    获取某个健的值

    contacts["小明"]
    补充：

    列表属于可变类型，不能作为键

    ![image-20240713131901300](python%E5%9F%BA%E7%A1%80.assets/image-20240713131901300.png

    元组：与列表的区别是元组使用圆括号。元组可作为字典的键

    元组不可变，添加删除等操作通通不可使用

    example_tuple = ("键盘"，键帽)
    字典是可变的：

    往字典中添加新的键值对，更新值的操作同样如此

    contacts["小丽"]  ="12345678901"

    如果想知道某个键是否存在

    "小明" in contacts
#存在返回True  （注意在python中布尔类型是大写）

    删除键值对

    del contacts["小明"]
len(contacts)  #查询字典中有多少键值对

for staff_id,temperature in temperature_dict.itens():

    if temperature >= 38:
        print(staff_id)
